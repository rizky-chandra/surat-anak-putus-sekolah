# Suara Anak Putus Sekolah

> **"Konstitusi menjamin Wajib Belajar 9 Tahun. Artinya Tuhan pun tidak bisa melarang."**

## 📍 Tentang Proyek Ini

Ini adalah suara langsung dari seorang anak di **Kotapinang, Labuhanbatu Selatan, Sumatera Utara** yang berhenti sekolah karena faktor ekonomi keluarga. Meskipun dicap sebagai "anak nakal" karena pernah merokok dan bolos, akar masalah sebenarnya adalah **ketidakmampuan membayar biaya sekolah**.

Anak ini memiliki argumen yang sangat kuat berdasarkan **UUD 1945 Pasal 31 Ayat 2**:
> "Setiap warga negara wajib mengikuti pendidikan dasar dan pemerintah wajib membiayainya."

Website ini adalah bentuk "surat kaleng" digital yang awalnya ditujukan untuk orang tua, kemudian berkembang menjadi halaman interaktif yang bisa disebarkan ke masyarakat luas untuk meminta pertolongan.

## 🎯 Tujuan

1. Menyuarakan hak konstitusional anak untuk mendapatkan pendidikan
2. Membuka mata masyarakat tentang realita anak putus sekolah karena faktor ekonomi
3. Mencari pertolongan dari berbagai pihak (Kepling, Dinas Pendidikan, Masyarakat, Media, Polisi, Kejaksaan)
4. Menunjukkan bahwa di balik label "anak nakal" ada masalah sistemik yang lebih dalam

## 🚀 Menjalankan Proyek

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Jalankan development server:
   ```bash
   npm run dev
   ```

3. Build untuk production:
   ```bash
   npm run build
   ```
